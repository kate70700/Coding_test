# 강아지 마법사의 꿈

기억 잃은 강아지 마법사가 꿈 속에서 잃어버린 기억 조각들을 모으며 자신의 아픈 과거를 치유하고, 결국 주인과 행복하게 재회하는 힐링 게임입니다.

## 게임 소개

이 게임은 유기견들의 다양한 사연을 바탕으로, 플레이어가 응원의 문장을 작성해 강아지들이 행복한 길로 나아가도록 돕는 따뜻한 시스템을 담고 있습니다.

### 게임 플레이

1. **기억 조각 탐색**: 꿈속 공간에서 흩어진 기억 조각(방치, 학대, 유기)을 선택합니다.
2. **스토리 감상**: 각 기억 조각에 담긴 강아지의 사연을 읽습니다.
3. **위로 문장 입력**: 상황에 맞는 위로와 응원의 문장을 직접 입력합니다.
4. **기억 회복**: 모든 기억 조각을 획득하면 강아지 마법사의 기억이 완전히 회복됩니다.
5. **엔딩**: 주인과의 따뜻한 재회 엔딩을 감상합니다.

## 실행 방법

### 필요 조건
- Python 3.x
- pygame

### 설치
```bash
pip install pygame
```

### 실행
```bash
python game.py
```

## 조작 방법

- **마우스 클릭**: 메뉴 선택, 기억 조각 선택, 화면 진행
- **키보드 입력**: 위로 문장 작성
- **Enter 키**: 문장 제출

## 주요 효과

1. 짧은 플레이타임으로 바쁜 일상 속에서도 가볍게 즐기며 힐링
2. 유기견 문제에 대한 자연스러운 인식 제고
3. 직접 위로의 말을 작성하며 공감과 따뜻한 감정 공유

## 기획 의도

유기견들의 이야기를 통해 플레이어가 단순한 게임 경험을 넘어 그들의 아픔을 이해하고, 입양의 필요성을 알리는 계기가 되길 바랍니다.
