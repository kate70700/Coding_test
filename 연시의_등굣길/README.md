# 연시의 등굣길

미연시(미소녀 연애 시뮬레이션) 게임 프로젝트입니다.

## 현재 구현 상태

- ✅ 시작 화면
- 🚧 게임 화면 (추후 구현 예정)

## 프로젝트 구조

```
연시의_등굣길/
├── index.html      # 메인 HTML 파일
├── style.css       # 스타일 시트
├── script.js       # JavaScript 로직
├── images/         # 이미지 파일 폴더
│   └── 연시의_등굣길.png
├── videos/         # 영상 파일 폴더
│   └── 연시의_등굣길.mp4 (또는 .webm)
└── README.md
```

## 설치 및 실행 방법

### 1. 이미지 파일 추가 (필수)

`images/` 폴더에 게임 타이틀 이미지를 넣어주세요:

- **연시의_등굣길.png** (또는 .jpg) - 게임 타이틀 로고 이미지

지원 형식: PNG, JPG, GIF 등

### 2. 영상 파일 추가 (선택사항)

`videos/` 폴더에 **"연시의_등굣길.mp4"** 파일을 넣어주세요.

지원 형식:
- MP4 형식: `연시의_등굣길.mp4`
- WebM 형식: `연시의_등굣길.webm`

> 영상이 없어도 게임은 실행되며, 기본 그라데이션 배경이 표시됩니다.

### 3. 게임 실행

브라우저에서 `index.html` 파일을 여세요.

**방법 1: 파일 직접 열기**
- `index.html` 파일을 더블클릭하거나
- 브라우저로 드래그 앤 드롭

**방법 2: VS Code의 Live Server 사용 (권장)**
- VS Code에서 "Live Server" 확장 설치
- `index.html` 파일에서 우클릭 → "Open with Live Server"

**방법 3: 로컬 서버 사용**
```bash
# Python이 설치되어 있다면
python -m http.server 8000

# 또는 Python 3
python3 -m http.server 8000

# 그 후 브라우저에서 http://localhost:8000 접속
```

### 4. 게임 시작

- **마우스 클릭**: 시작 화면의 아무 곳이나 클릭
- **키보드**: 스페이스바 또는 엔터키를 누르세요

## 기능

### 시작 화면
- 배경 영상 자동 재생 (loop)
- 게임 타이틀 이미지 표시 (페이드 인 애니메이션)
- "클릭하여 시작" 텍스트 표시 (깜빡이는 애니메이션)
- 클릭/키보드 입력 시 게임 화면으로 전환 (페이드 효과)
- 타이틀 이미지에 그림자 효과 적용

### 향후 구현 예정
- 스토리 시스템
- 캐릭터 대화 시스템
- 선택지 시스템
- 캐릭터 일러스트 표시
- 배경음악 및 효과음
- 세이브/로드 기능

## 기술 스택

- HTML5
- CSS3 (애니메이션, 반응형 디자인)
- Vanilla JavaScript (ES6+)

## 주의사항

- 일부 브라우저에서는 보안 정책으로 인해 영상 자동 재생이 차단될 수 있습니다.
- 로컬 서버를 사용하면 이러한 문제를 피할 수 있습니다.
