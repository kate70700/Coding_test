# 이미지 파일 가이드

이 폴더에 캐릭터 이미지를 배치하세요 (선택사항).

## 필요한 파일 목록

- `rabbit.png` - 토끼 캐릭터 이미지
- `cat.png` - 고양이 캐릭터 이미지
- `dog.png` - 강아지 캐릭터 이미지
- `bear.png` - 곰 캐릭터 이미지
- `bird.png` - 새 캐릭터 이미지
- `penguin.png` - 펭귄 캐릭터 이미지
- `panda.png` - 판다 캐릭터 이미지
- `fox.png` - 여우 캐릭터 이미지

## 이미지 요구사항

- **형식**: PNG (투명 배경 권장)
- **크기**: 200x200 픽셀 이상
- **스타일**: 귀여운 캐릭터 스타일

## 참고

이미지가 없어도 프로그램은 정상 작동합니다.
이미지가 없을 경우 자동으로 귀여운 원형 캐릭터가 표시됩니다.
