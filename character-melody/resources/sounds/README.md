# 음원 파일 가이드

이 폴더에 캐릭터별 WAV 음원 파일을 배치하세요.

## 필요한 파일 목록

다음 파일들이 필요합니다:

- `rabbit.wav` - 토끼 캐릭터 음원
- `cat.wav` - 고양이 캐릭터 음원
- `dog.wav` - 강아지 캐릭터 음원
- `bear.wav` - 곰 캐릭터 음원
- `bird.wav` - 새 캐릭터 음원
- `penguin.wav` - 펭귄 캐릭터 음원
- `panda.wav` - 판다 캐릭터 음원
- `fox.wav` - 여우 캐릭터 음원

## 음원 요구사항

- **형식**: WAV (16-bit PCM)
- **샘플 레이트**: 44100 Hz 권장
- **길이**: 2-4초 (루프 가능한 음원)
- **볼륨**: 비슷한 레벨로 정규화

## 음원 추천

각 캐릭터에 적합한 음원 타입:

- **토끼**: 경쾌한 멜로디, 하이햇
- **고양이**: 부드러운 신스, 패드
- **강아지**: 리듬 베이스
- **곰**: 저음 베이스, 킥 드럼
- **새**: 하이 피치 멜로디, 플루트
- **펭귄**: 퍼커션, 스네어
- **판다**: 중음 멜로디
- **여우**: 보컬 샘플, 리드
