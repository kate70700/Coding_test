# 캐릭멜로디 (Character Melody) 🎵

귀여운 캐릭터들을 클릭하여 음악을 창작하는 인터랙티브 음악 프로그램입니다.

## 📌 프로젝트 개요

**제작자**: 위은서 (2407)
**제작일**: 2025.08.29
**버전**: 1.0

인크레디박스(Incredibox)에서 영감을 받아, 귀여운 캐릭터 디자인과 직관적인 인터페이스를 통해 누구나 쉽게 음악을 창작하고 즐길 수 있는 프로그램입니다.

## ✨ 주요 기능

- 🎭 **다양한 캐릭터**: 8개의 귀여운 캐릭터 (토끼, 고양이, 강아지, 곰, 새, 펭귄, 판다, 여우)
- 🎵 **음악 창작**: 캐릭터를 클릭하여 각각의 고유한 소리 재생
- 🎼 **실시간 믹싱**: 여러 캐릭터를 동시에 클릭하여 독창적인 음악 조합 생성
- 🎨 **시각적 피드백**: 재생 중인 캐릭터는 색상과 상태가 변경되어 시각적으로 확인 가능
- 🛑 **간편한 컨트롤**: '모두 정지' 버튼으로 모든 소리를 한 번에 중지

## 🎮 사용 방법

1. 프로그램을 실행하면 메인 화면에 8개의 캐릭터가 표시됩니다.
2. 원하는 캐릭터를 클릭하면 해당 캐릭터의 소리가 반복 재생됩니다.
3. 여러 캐릭터를 클릭하여 동시에 재생하면 음악이 조합됩니다.
4. 다시 클릭하면 해당 캐릭터의 소리가 정지됩니다.
5. '모두 정지' 버튼을 클릭하면 모든 소리가 중지됩니다.

## 🚀 실행 방법

### 필요 사항
- Java 8 이상
- WAV 형식의 음원 파일

### 컴파일 및 실행

```bash
# 1. 프로젝트 디렉토리로 이동
cd 캐릭멜로디

# 2. 소스 파일 컴파일
javac -d bin -encoding UTF-8 src/main/*.java

# 3. 실행
java -cp bin main.CharacterMelody
```

### IDE에서 실행
1. IntelliJ IDEA, Eclipse 등의 IDE에서 프로젝트를 엽니다.
2. `src/main/CharacterMelody.java` 파일을 열고 실행합니다.

## 🎵 음원 준비하기

프로그램이 정상적으로 작동하려면 음원 파일이 필요합니다.

### 음원 준비 방법

1. **WAV 파일 준비**: 각 캐릭터에 맞는 WAV 형식의 음원을 준비합니다.
   - 비트, 멜로디, 베이스, 보컬 등 다양한 소리 요소
   - 루프 가능한 음원 권장 (2~4초 길이)

2. **파일 배치**: `resources/sounds/` 폴더에 다음과 같이 배치합니다:
   ```
   resources/sounds/rabbit.wav
   resources/sounds/cat.wav
   resources/sounds/dog.wav
   resources/sounds/bear.wav
   resources/sounds/bird.wav
   resources/sounds/penguin.wav
   resources/sounds/panda.wav
   resources/sounds/fox.wav
   ```

3. **음원 추천**:
   - 무료 음원 사이트: Freesound.org, Zapsplat.com
   - 직접 제작: Audacity, GarageBand 등의 프로그램 사용
   - 각 음원의 볼륨을 비슷하게 조정하면 더 좋은 믹싱 효과

## 🎨 이미지 준비하기 (선택사항)

캐릭터 이미지를 추가하면 더 귀여운 UI를 구현할 수 있습니다.

1. PNG 형식의 캐릭터 이미지 준비 (투명 배경 권장)
2. `resources/images/` 폴더에 배치
3. 이미지가 없으면 자동으로 귀여운 원형 캐릭터로 대체됩니다.
